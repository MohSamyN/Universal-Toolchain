# Universal-Toolchain

A minimal universal CLI tool to build and run projects across multiple languages using a simple folder-based `build.tc` configuration.

---

## Features

- Recursive folder support
- Auto language detection
- Supports:
  - C
  - C++
  - Python
  - Java
- Zero configuration complexity
- Single-file CLI tool

---
